1. создаем папку 1structures, туда помещаем данные 
о структуре в формате программы VASP (файлы POSCAR)

2. создаем папку 2enkpoint, в неё помещаем скрипт
двойного запуска (оптимизация, затем статический расчет).
Сам скрипт: run2OptAndStat
этот скрипта пользуется файлами
tINCAR - для оптимизации
tKPOINTS - для оптимизации
INCARstat - для статического расчета
+ также используется скрипт potcar_gen

3. в результате основной расчет будет лежать в папке stat
туда надо скопировать скрипты: readEIGENVAL.py и share
сначала выполнить скрипт readEIGENVAL.py, и
в результате выполнения появится файл bands.txt
далее выполнить share, после чего появиться папка с файлами

Эти файлы надо нарисовать с помощью gnuplot, используя команды:

band(n)=sprintf("%d",n)
plot for [i=0:23] band(i) with lines

Рисовать только 2 и 3 столбцы из файлов!
